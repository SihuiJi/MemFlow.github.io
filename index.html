Long Context Memory Showcases

MemFlow maintains long-range consistency and semantic coherence in challenging cases, including camera movement, character insertion, and scene switching. In the first 0–10 seconds of the long video, the scene contains only a single subject. Between 10–20 seconds, a new subject enters the scenario. From 20–40 seconds, multiple subjects interact while each maintains its own visual consistency—exhibiting neither drifting nor identity confusion. Between 40–50 seconds, the camera angle changes, with some subjects temporarily moving out of view. At 50–60 seconds, the viewpoint returns to its original position, and the subjects still preserve the same characteristics they had before the camera switch, highlighting our advantage in long context memory.

Overall Framework

Our Framework is designed for interactive long video generation with both long-term consistency and efficiency. In autoregressive generation, we conduct memory retrieval, updating, and selection sequentially before finally use it for synthesizing the incoming video chunk. Given the current prompt and preceding KV cache memory bank, we first use textual token from the prompt to query the memory and the previous chunk to retrieve semantically aligned KV cache. After adding KV cache of previous chunk after merging to inject the latest context, the memory is updated for current chunk generation. The updated memory is then combined with "Frame Sink"—KV cache from the first chunk—for the following Sparse Memory Activation (SMA). SMA filters the most relevant context for attention computation, improving the generation efficiency without sacrificing visual quality.
